## Quick start

### Requirements
* [Node.js (v4.2.x)](https://nodejs.org/en/download/)
* [node-mysql](https://github.com/felixge/node-mysql)

### Windows
* Make sure you have 'node.js' installed

* Clone the repo:
```bash
> git clone https://github.com/SWMEM-ONLINE/ERP-SYSTEM.git
> cd ERP-SYSTEM
```

* Run the server:
```bash
> node app.js
```
